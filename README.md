A simple Meteor Javascript performance timer

## Add to project
```
meteor add anthonyastige:performance-timer
```

## Add to top
```
import { PT } from 'meteor/anthonyastige:performance-timer'
```

## Run these

```
PT.a()	// PT.start
PT.z()	// PT.stop
PT.q()	// PT.quick
PT.qz()	// PT.quickStop
```
